Symbolism

  • Hexagon Shape: Represents relationship to Envoy Proxy, the underlying technology
  • Four Directional Arrows: Represent API request flows through the gateway
  • Northbound Ingress (‚Üì): External API traffic flowing into the gateway. This is the incoming traffic from external sources such as external clients or other services.
  • Southbound Egress (‚Üì): External API traffic flowing out of the gateway. This is the outgoing traffic to external destinations such as 3rd party services or other external systems.
  • Westbound Ingress (‚Üí): Internal API traffic flowing into the gateway from internal network/organization. This is the incoming traffic from internal sources such as internal services or other internal systems.
  • Eastbound Egress (‚Üí): Internal API traffic flowing out of the gateway to another internal service. This is the outgoing traffic to internal destinations such as other internal services or other internal systems.
  • Central Gateway: Single control point managing all API traffic in all directions. This is the central point of the API management system that is responsible for managing all API traffic in all directions.
