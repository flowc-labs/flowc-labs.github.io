# flowc Logo Design

## Design Concept

The flowc logo is inspired by clean, minimalist CNCF project branding, featuring a dynamic radial design that represents the flow and control of API traffic. The radiating segments symbolize the omnidirectional nature of API management and the flow of data through the control plane.

### Design Elements

#### 1. **Radial Icon Design**
- Dynamic radial pattern with 8 radiating segments
- Represents the omnidirectional flow of API traffic
- Uses vibrant purple-to-blue gradient (#DD37E0 to #0BA7FF)
- Symbolizes the central control point from which API management flows outward
- Clean, modern geometric design that scales beautifully at any size

#### 2. **Gradient Color System**
- **Primary Gradient**: Purple (#DD37E0) to Blue (#0BA7FF)
- Represents the flow from control (purple) to execution (blue)
- Applied consistently across icon and typography
- Creates visual harmony and brand recognition

#### 3. **Typography**
- Bold, sans-serif lettering: "FLOWC"
- Full gradient text matching the icon colors
- Letter spacing for modern, tech-forward appearance
- Represents the complete API management platform

### Color Palette

- **Primary Purple**: #DD37E0 (start of gradient)
- **Primary Blue**: #0BA7FF (end of gradient)
- **Gradient**: Linear gradient from purple to blue
- All colors create a cohesive, modern brand identity

### Design Philosophy

**Simplicity & Clarity**
- Clean radial geometric design
- No unnecessary details
- Instantly recognizable at any size
- Modern, tech-forward aesthetic

**Symbolism**
- Radial pattern = Centralized control with omnidirectional flow
- Gradient = Flow from control to execution
- Eight segments = Comprehensive coverage and completeness
- Dynamic design = Active API management and real-time control

**Professional Aesthetic**
- CNCF landscape compatible
- Cloud-native technology branding
- Enterprise-grade appearance
- Scalable and versatile

### File Variants

1. **flowc-logo.svg** (240×320)
   - Full logo with text
   - Use for: Documentation covers, presentations, marketing

2. **flowc-icon.svg** (200×200)
   - Icon-only version (no text)
   - Use for: Navbar, favicon, app icons, small displays

3. **flowc-logo-horizontal.svg** (400×120)
   - Horizontal layout with icon and text side-by-side
   - Use for: Website headers, banners, email signatures

### Usage Guidelines

#### Sizing
- **Minimum icon size**: 24px (icon remains legible)
- **Recommended sizes**: 32px, 48px, 64px (navbar), 128px, 256px
- Vector format ensures perfect scaling at any size

#### Spacing
- Maintain minimum clear space of 12px around the logo
- For icon version: 8px minimum clear space

#### Backgrounds
- **Light backgrounds**: Full color version (default)
- **Dark backgrounds**: Logo maintains visibility with its gradient design
- **Purple gradient**: Perfect match with hero banner

#### Colors - Do's
✅ Use the provided gradient as specified (#DD37E0 to #0BA7FF)
✅ Maintain consistent gradient application across icon and text
✅ Preserve the vibrant purple-to-blue transition

#### Colors - Don'ts
❌ Don't change colors to other hues
❌ Don't use single flat colors instead of gradients
❌ Don't reduce opacity below legibility threshold
❌ Don't add drop shadows or effects

### Technical Specifications

- **Format**: SVG (Scalable Vector Graphics)
- **Color space**: RGB
- **Transparency**: Supported
- **Gradients**: Linear gradients with defined stops
- **Stroke width**: Optimized for clarity
- **Text**: Converted to paths for consistent rendering
